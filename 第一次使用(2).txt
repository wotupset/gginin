第一次使用Git要先設定使用者資料(可以隨便打)<br>
這部份資料會存在"C:\Documents and Settings\使用者名稱\.gitconfig"<br>
只能用git bash輸入<br>
git config --global user.email "your@email.com"<br>
git config --global user.name "IRWINNER"<br>
<img border="1" src="http://i.imgur.com/MMA3OU2.png"><br>
輸入完後 就可以執行Git GUI<br>
<br>
第一次執行的Git GUI 畫面<br>
<img border="1" src="http://i.imgur.com/dYdPyb6.png"><br>
<dl>
<dt>create new repository</dt>
<dd>建立新repo</dd>
<dt>clone existing repository</dt>
<dd>複製已有的repo</dd>
<dt>open exist repository</dt>
<dd>開啟已有的repo</dd>
</dl>

<dt>rescan</dt>
<dd>掃瞄資料夾的檔案變動</dd>
